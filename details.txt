Problem statement :
You are supposed to create an idea service. Users can call this idea service to add,update,and search
and delete any idea.

{

    "id": 124,
    "idea_name" : "great idea",
    "author_name" : "Vishwa Mohan",
    "idea_description" : "description"

}

Following API should be implemented :

POST : /idea_app/v1/ideas/

GET : /idea_app/v1/ideas/

GET : /idea_app/v1/ideas/{id}

PUT : /idea_app/v1/ideas/{id}

DELETE : /idea_app/v1/ideas/{id}
